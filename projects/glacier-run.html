Glacier Run

A 2D side-scrolling platformer set across treacherous icy glacier levels — leap between platforms, dodge obstacles, and race to the finish. One wrong step and the level restarts instantly, making every successful run that much more satisfying.

C# Unity 2D Platformer WebGL

Glacier Run is a 2D side-scrolling platformer built in Unity, set across a series of icy, glacier-themed obstacle courses. Players guide a character through increasingly challenging levels — jumping between floating ice platforms, avoiding spike traps, and navigating narrow gaps that demand precision and timing.

The defining mechanic is simple and ruthless: fall off a platform or hit an obstacle and the level immediately restarts from the beginning. There are no checkpoints within a level — every attempt is a clean run from start to finish. This design keeps the gameplay loop tight, intense, and deeply rewarding when you finally nail a clean clear.

Core Gameplay

  • Run & Jump: Responsive side-scrolling movement with snappy jump mechanics — precise control is key to clearing gaps and obstacle sequences.
  • Instant Restart on Fall: Any fall or collision with a hazard immediately restarts the current level from scratch — no lives lost, just retry immediately, keeping frustration low and motivation high.
  • Level Completion: Reach the end flag of each level to advance to the next — levels are completed in sequence, each building on the skills required by the previous.
  • Icy Physics: The glacier environment introduces subtle momentum and slip physics — characters slide slightly on ice surfaces, adding an extra layer of challenge to precise landings.
  • Progressive Difficulty: Early levels establish basic jump and run mechanics; later levels introduce moving platforms, longer gaps, spike corridors, and timing-based hazards.

Level Design

Each level in Glacier Run is a self-contained obstacle course designed around a central challenge — teaching a new skill, then testing it under pressure. The icy glacier aesthetic is consistent throughout, with layered parallax backgrounds giving depth to the frozen world.

  • Multiple levels with increasing obstacle density and complexity
  • Static and moving ice platforms — some shift horizontally or vertically
  • Spike traps at the base of gaps — fall in and restart immediately
  • Narrow ledges requiring precise landing and careful movement
  • Timing-based sections where platforms disappear and reappear
  • Icy glacier visual theme with layered parallax background scrolling
  • Level end flag clearly visible, giving players a clear target each run

Key Features

  • Tight, responsive 2D platformer controls optimised for precision movement
  • Instant level restart on fall or obstacle collision — no loading screens
  • Glacier/ice visual aesthetic with smooth parallax background
  • Progressive level difficulty — beginner-friendly start, demanding later stages
  • Moving and static ice platforms with Unity 2D physics integration
  • Smooth character animations: run, jump, fall, and death sequences
  • Built with Unity 2D Tilemap system for clean level construction
  • WebGL build — playable directly in the browser with no installation
