Operation CrossFire

A modular first-person shooter system with advanced weapon mechanics, shooting modes, and a dynamic HUD — built in Unity Engine.

C# Unity Engine Game Development

Developed a modular first-person shooter system featuring weapon pickup, weapon switching, shooting modes (Single/Burst/Auto), recoil, reloading, and ammo management. Implemented optimized weapon state handling, smooth animations, and sound integration using enums and structured manager classes.

Built a dynamic HUD system to update weapon icons, ammo counts, and UI elements in real time. The project demonstrates advanced game architecture with modular, reusable components that can be extended for larger FPS game projects.

Key Features

  • Modular weapon system — pickup, switch, and drop weapons dynamically
  • Multiple shooting modes: Single, Burst, and Auto fire
  • Realistic recoil and weapon sway mechanics
  • Reloading system with ammo management and magazine tracking
  • Optimized weapon state handling using enums and structured manager classes
  • Smooth animations and sound integration for immersive gameplay
  • Dynamic HUD displaying weapon icons, ammo counts, and UI elements in real time
  • Clean, modular architecture designed for extensibility
