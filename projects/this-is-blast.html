This Is Blast (Clone)

A 2D match-and-blast puzzle game built in Unity — tap groups of same-coloured blocks to clear them, watch gravity pull the remaining blocks down, and chain reactions to maximise your score.

C# Unity 2D Puzzle Game Mobile

This Is Blast (Clone) is a mobile-style 2D puzzle game developed in Unity, inspired by the popular match-and-blast genre. The core mechanic is elegantly simple — tap any group of two or more adjacent same-coloured blocks to blast them off the board. Gravity immediately takes over, pulling all remaining blocks downward to fill the gaps and create new adjacencies for potential chain reactions.

The game is built around a colour-coded grid system with real-time physics-style block falling, a dynamic scoring engine, and progressively challenging level objectives — making each puzzle both accessible and strategically deep.

Core Gameplay Mechanics

  • Tap to Blast: Tap any block that is part of a group of 2 or more adjacent same-coloured blocks — the entire connected group disappears instantly.
  • Gravity System: After every blast, all blocks above the cleared area fall straight down, filling empty columns from the bottom up — creating natural chain-reaction opportunities.
  • Colour Types: Four distinct block colours — Red, Blue, Green, and Orange — randomly distributed across the grid, keeping every level unpredictable.
  • Chain Reactions: Gravity-driven block settling can cause newly adjacent same-colour groups to form, rewarding strategic tapping order over brute-force clearing.
  • Score Multipliers: Larger group blasts award exponentially higher points — clearing 10+ blocks in a single tap triggers a combo bonus.
  • Level Objectives: Each level sets a target — clear a specific colour, reach a score, or clear the board — within a limited number of moves.

Grid & Block System

The game board is a dynamically sized grid (default 10×10) populated with randomly distributed coloured blocks at level start. The board manager tracks adjacency in real time, highlights tappable groups on touch, and triggers the fall animation whenever a gap is created.

  • Grid dimensions configurable per level (e.g. 8×8 up to 12×12)
  • Flood-fill algorithm detects all connected same-colour neighbours instantly
  • Animated block fall using Unity's DOTween for smooth gravity response
  • New blocks spawn at the top of each column to refill the board (endless mode)
  • Isolated single blocks (no neighbours) are marked with a visual indicator
  • Shuffle mechanic triggers automatically if no valid moves remain

Key Features

  • Smooth touch-responsive block selection with visual group highlight on press
  • Real-time gravity simulation — blocks fall and settle with polished animations
  • Four colour types (Red, Blue, Green, Orange) with distinct particle blast effects
  • Score system with combo multipliers for large group clears
  • Multiple level objectives: colour clear, score target, board clear
  • Progressive difficulty — denser same-colour clusters decrease as levels advance
  • Endless mode with continuous block refill for score-chasing gameplay
  • Clean, vibrant 2D art style optimised for mobile screens
  • Built with Unity 2D — optimised for Android performance
