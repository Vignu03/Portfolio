Flap with Caution

A light-hearted, fun tap-to-fly game where a clueless bird dodges obstacles and stays airborne — built in Unity Engine.

C# Unity Engine Game Development

Designed and developed a light-hearted tap-to-fly game, Flap With Caution, where a clueless bird tries to dodge moving obstacles and stay airborne. Built smooth physics-based controls, collision handling, scoring, and overall game flow in Unity.

Focused on creating a polished, fun, and easy-to-pick-up gameplay experience. The game features increasing difficulty, responsive controls, and a clean visual style that makes it engaging for players of all ages.

Key Features

  • Smooth physics-based tap-to-fly controls for intuitive gameplay
  • Dynamic obstacle generation with increasing difficulty
  • Collision detection and handling system
  • Real-time scoring system that tracks player progress
  • Polished game flow — start screen, gameplay, and game over
  • Clean visual design with engaging animations
  • Easy-to-pick-up gameplay mechanics for casual gaming
