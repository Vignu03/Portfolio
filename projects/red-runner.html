Red Runner

A fast-paced 2D side-scrolling runner built in Unity — guide a red character through an endless gauntlet of obstacles. One hit and the game restarts instantly, keeping every attempt sharp, quick, and addictive.

C# Unity 2D Endless Runner WebGL

Red Runner is a 2D side-scrolling runner game developed in Unity. The player controls a distinctive red character that automatically moves forward through procedurally generated terrain — jumping, sliding, and dodging through a relentless stream of obstacles, gaps, and hazards.

The core loop is ruthlessly simple: survive as far as you can. Any collision with an obstacle or fall into a gap triggers an instant death and the game restarts immediately from the beginning. There are no checkpoints, no extra lives — just pure reflex-based gameplay that rewards timing and pattern recognition.

Core Gameplay Mechanics

  • Auto-Run: The red character continuously moves forward at an increasing pace — the player's job is to react to incoming obstacles using precisely timed jumps and slides.
  • Jump & Slide: Two primary actions — jump to clear gaps and elevated obstacles, slide/duck to avoid overhead barriers. Timing is everything.
  • Instant Death & Restart: Any contact with an obstacle or falling off the terrain kills the character instantly. The game restarts immediately with zero loading time — keeping the retry loop tight and addictive.
  • Increasing Speed: The longer you survive, the faster the character moves — creating a natural difficulty curve that keeps even skilled players challenged.
  • Score System: Distance survived serves as the player's score. Each run pushes players to beat their personal best, driving replayability.

Obstacle Types

The game features a variety of obstacles designed to test different reaction skills. Obstacles are procedurally placed to create unique runs every time, preventing memorisation and keeping gameplay fresh.

  • Ground spikes — sharp obstacles requiring well-timed jumps
  • Elevated barriers — overhead hazards that require sliding underneath
  • Gap pits — breaks in the terrain that the character must jump across
  • Moving obstacles — horizontally shifting barriers that demand timing precision
  • Combo sequences — rapid obstacle chains requiring alternating jumps and slides
  • Procedural placement ensures no two runs are identical

Key Features

  • Fast-paced 2D side-scrolling runner with smooth, responsive controls
  • Distinctive red character with fluid run, jump, slide, and death animations
  • Instant death and restart — zero downtime between attempts
  • Progressive speed increase for natural difficulty scaling
  • Procedurally generated obstacle sequences for infinite replayability
  • Distance-based scoring with personal best tracking
  • Polished 2D art with parallax scrolling backgrounds
  • Particle effects on death and obstacle collisions
  • WebGL build — playable directly in the browser with no installation
  • Built with Unity 2D physics and animation system
